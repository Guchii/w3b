A set of simple and useful hooks for react facilitating use of the Ethereum Provider API (under development).

uses <b>ethers, zustand, and immer.js</b>

### Installation

```sh
npm install @guchii/w3b
```

```sh
yarn add @guchii/w3b
```

```sh
pnpm add @guchii/w3b
```
